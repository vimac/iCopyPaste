A visual SQL Map code generator, hoping to save your time when coding to access the database.

Features

  • Supports all major OS platforms: Windows / macOS / Linux
  • Supports MyBatis(Java) & MySpot(PHP)
  • Project structure generation
  • Query statement visualize generation
